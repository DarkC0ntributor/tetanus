## Description
Change the agent's current working directory.

### Parameters
`directory`
 * Path of the directory to navigate to

## Usage
```
cd [path]
```
```
cd -directory [path]
```

### Examples
```
cd /tmp
```
```
cd C:\Users\Public
```
```
cd -directory ../../Documents
```

### Popup
Command supports using the Mythic UI popup for entering parameters.
