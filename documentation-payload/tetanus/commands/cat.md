## Description
Output the contents of a file.

### Parameters
`file`
 * Path to the file

## Usage
```
cat [path]
```
```
cat -file [path]
```

### Examples
```
cat /etc/passwd
```
```
cat C:\Users\windows\Desktop\file.txt
```
```
cat -file ../file.txt
```

### Popup
Command supports using the Mythic UI popup for entering parameters.
