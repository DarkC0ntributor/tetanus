## Description
Remove an environment variable

### Parameters
`variable`
 * Name of the environment variable to remove

## Usage
```
unsetenv [variable]
```
```
unsetenv -variable [variable]
```

### Examples
```
unsetenv foo
```
```
unsetenv -variable foo
```
